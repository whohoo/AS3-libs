InheritanceAddRemoveEffectTargetFilter EffectTargetFilter

AddRemoveEffectTargetFilter is a subclass of EffectTargetFilter that handles the logic for filtering targets that have been added or removed as children to a container. If you set the Effect.filter property to "add" or "remove", then one of these is used.
