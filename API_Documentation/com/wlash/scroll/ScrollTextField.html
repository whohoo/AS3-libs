Classpublic final class ScrollTextField
InheritanceScrollTextField SimpleScrollBar flash.display.Sprite

滚动TextField框内容.

需在参数面版中定义scrollTarget为文本框对象。

参数面版：
  • direction: 水平方向或垂直方向滚动文本框。
    • 其它参数请参考父类ScrollBar
