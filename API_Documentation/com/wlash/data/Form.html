Class FormMethods
 
Packagecom.wlash.data
Classpublic class Form

一个实用的表单提交检查类.

可定义输入文本框的顺序，对于单行输入框，按下回车(Enter)键时，可自动跳到下一 文本框中。最后一个文本框中按下回画键，相当于提交数据。

还可对邮件，手机，身份证号，日期等数据进行检验。

checkEmail()method 






public static function checkEmail(mailStr:String):Boolean

  check email.
  
建议用 [restrict = "A-Za-z0-9\\-_.@"]来限制文本框的输入.
  只对输入的邮件的大概结构做判断,而不是全面合法性判断.
  要求邮件地址:只能以字母或数字开头,可以有(_或-)连继的存在,如a---b@live.com
  但不能有a...b@live.com的存在,因为不合法,也不能是abc.@live.com
  可以是以IP地址结尾的电子邮件,如abc@123.22.34.32
  

  Parameters




mailStr:String — eamil address
  






Returns




Boolean — true or false
  
  






Example

sample code:
  
  Form.checkEmail("whohoo@21cn.com");//return true
